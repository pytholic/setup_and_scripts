# System setup

Install the following.
* Ubuntu
* Sublime text
* VS Code
  * Markdown All in One
  * Jupyter
  * python
  * IntelliCode
  * Remote - SSH
  * Git Graph
  * Pylance
* SmarGit
* Nvidia cuda toolkit
* Anaconda
* fzf
```shell
sudo apt-get install fzf

Add to bash:
nano ~/.bashrc
. /usr/share/doc/fzf/examples/key-bindings.bash
```
